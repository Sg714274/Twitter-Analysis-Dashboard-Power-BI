Key Insights Extracted

Tweet activity dips significantly on weekends.

Highest impressions occur mid-week (Tuesday to Thursday).

URL clicks show a strong correlation with certain types of tweets, especially informational or promotional ones.

Engagement rate remains consistent despite drop in impressions.

Media views outperform engagement rates on weekends.

Hashtag clicks are less compared to profile and URL clicks, indicating users are more likely to explore profiles or links than hashtags.
